Hash

hash는 쌍으로 이루어진 자료구조이며 다음과 같은 특징을 가진다.

  1. 하나의 Key는 오브젝트명과 하나 이상의 필드 값을 콜론(:) 기호로 결합하여 표현할 수 있다
  2. 문자값을 저장할 때는 " "를 사용하며 숫자 값을 저장할 때는 필요없다
  3. 기본적으로 필드 개수의 제한은 없다

Command

  • HMSET [key] [field1] [value1] [field2] [value2] …
  • HGET [key] [field]
  • HGETALL [key]
  • HEXISTS [key] [field]
  • HDEL [key] [field]
  • HKEYS [key] : key에 대한 모든 field를 가져온다
  • HVALS [key] : key에 대한 모든 value를 가져온다
  • HLEN [key]
  • HMGET [key] [field1] [field2] …
