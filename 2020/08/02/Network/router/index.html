Router

라우터는 Path Determination과 Switching을 수행한다.

Static Routing

스태틱 라우팅은 관리자가 라우터에 목적지 별로 경로를 삽입해준다. 스태틱 라우팅을 수행하면 라우터의 메모리를 비롯한 성능면에서 이점이 존재한다. 하지만 목적지의 라우터가 다운된 상태라면 해당 라우터로 스위칭되는 패킷은 전송되지 못하는 단점이 있다.

Dynamic Routing

다이나믹 라우팅은 경로를 삽입해줄 필요가 없다. RIP나 IGRP와 같은 라우팅 프로토콜을 사용해 경로를 계산하고 유효한 경로를 찾아낸다. 그렇기 때문에 다양한 상황에서도 자동으로 대응이 가능하다.

Routing Table

라우팅 테이블은 라우터가 패킷을 전송할 때 목적지와 목적지를 가려면 어떤 인터페이스를 통해야하는지에 대한 정보를 담고 있다. 라우팅 테이블은 전원을 종료하면 지워지게 된다.

Autonomous System

AS는 하나의 네트워크 관리자에 의해서 관리되는 라우터 그룹이다. AS내에 속해있는 라우터는 자신의 네트워크 내에 존재하는 라우터의 존재만 알면 되고 존재하지 않는다면 Autonomous System Boundary Router로 외부로 전송하면 된다.
