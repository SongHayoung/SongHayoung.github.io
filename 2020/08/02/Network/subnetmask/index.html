Subnet Mask

서브넷 마스크는 주어진 IP 주소를 네트워크 환경에 맞게 나누어 주기 위해서 씌워주는 이진수의 조합이다. 즉 서브넷 마스크란 IP 주소를 가지고 어디까지가 네트워크 부 이고 어디까지가 호스트 부 인지 나타내는 역할을 수행한다. 서브넷 마스크를 이진수로 표현하고 1로 표현된 부분이 네트워크 부 이고 0으로 표현된 부분이 호스트 부 이다.

서브넷 마스크를 사용하면 커다란 네트워크를 작은 단위의 네트워크로 나누어 쪼갤 수 있다. 즉 서브넷은 하나의 독립된 네트워크가 된다.

서브넷 내의 첫번째 주소와 마지막 주소는 호스트 주소로 사용할 수 없다. 첫번째 주소는 네트워크 주소를 의미하고 마지막 주소는 브로드 캐스트 주소를 의미하기 때문이다.
