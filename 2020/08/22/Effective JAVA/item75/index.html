л“Өм–ҙк°Җл©°

мҳҲмҷёк°Җ л°ңмғқн–Ҳмқ„ л•Ң мҳҲмҷём—җ мӢӨнҢЁ мӣҗмқём—җ кҙҖн•ң м •ліҙлҘј к°ҖлҠҘн•ң л§Һмқҙ лӢҙм•„ л°ҳнҷҳн•ҳлҠ” мқјмқҖ м•„мЈј мӨ‘мҡ”н•ҳлӢӨ. мӢӨнҢЁ мҲңк°„мқ„ нҸ¬м°©н•ҳл Өл©ҙ л°ңмғқн•ң мҳҲмҷём—җ кҙҖм—¬лҗң лӘЁл“  л§Өк°ңліҖмҲҳмҷҖ н•„л“ңмқҳ к°’мқ„ мӢӨнҢЁ л©”м„ём§Җм—җ лӢҙм•„м•ј н•ңлӢӨ. мҳҲлҘј л“Өм–ҙ IndexOutOfBoundsExceptionмқҳ мғҒм„ё л©”м„ём§ҖлҠ” лІ”мң„мқҳ мөңмҶҢк°’, мөңлҢҖк°’, лІ”мң„лҘј лІ—м–ҙлӮң мқёлҚұмҠӨк°’мқ„ лӢҙм•„м•ј н•ңлӢӨ. мҠӨнғқ м¶”м Ғм—җлҠ” мҳҲмҷёк°Җ л°ңмғқн•ң нҢҢмқј мқҙлҰ„кіј мӨ„лІҲнҳёлҠ” л¬јлЎ  мҠӨнғқм—җм„ң нҳём¶ңн•ң лӢӨлҘё л©”мҶҢл“ңл“Өмқҳ нҢҢмқј мқҙлҰ„кіј мӨ„лІҲнҳёк№Ңм§Җ м •нҷ•нһҲ кё°лЎқлҗҳм–ҙ мһҲлҠ”кІҢ ліҙнҶөмқҙлӢӨ. к·ёл Үкё° л•Ңл¬ём—җ л¬ём„ңмҷҖ мҶҢмҠӨмҪ”л“ңм—җм„ң м–»мқ„ мҲҳ мһҲлҠ” м •ліҙлҠ” лӢҙм•„лҙҗм•ј кө°лҚ”лҚ”кё°к°Җ лҗ  лҝҗмқҙлӢӨ.

мҳҲмҷёмқҳ мғҒм„ё л©”м„ём§ҖмҷҖ мөңмў… мӮ¬мҡ©мһҗм—җкІҢ ліҙм—¬мӨ„ мҳӨлҘҳлҘј нҳјлҸҷн•ҙм„ңлҠ” м•ҲлҗңлӢӨ. мөңмў… мӮ¬мҡ©мһҗм—җкІҢлҠ” м Ғм Ҳн•ң м•ҲлӮҙ л©”м„ём§ҖлҘј, мҳҲмҷё л©”м„ём§ҖлҠ” к°ҖлҸ…м„ұліҙлӢӨлҠ” лӢҙкёҙ лӮҙмҡ©мқҳ м§Ҳмқҙ мӨ‘мҡ”н•ҳлӢӨ.

мҳҲмҷёлҠ” мӢӨнҢЁмҷҖ кҙҖл Ёлҗң м •ліҙлҘј м–»мқ„ мҲҳ мһҲлҠ” м ‘к·јмһҗ л©”мҶҢл“ңлҘј м Ғм ҲнһҲ м ңкіөн•ҳлҠ” кІғмқҙ мўӢлӢӨ. нҸ¬м°©н•ң мӢӨнҢЁ м •ліҙлҠ” мҳҲмҷё мғҒнҷ©мқ„ ліөкө¬н•ҳлҠ”лҚ° мң мҡ©н•ҳкё° л•Ңл¬ёмқҙлӢӨ.
