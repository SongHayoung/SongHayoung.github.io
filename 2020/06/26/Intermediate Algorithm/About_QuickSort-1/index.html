Quick Sort лҚ” м•Ңм•„ліҙкё° - 2

Implements

C / C++мқ„ кё°мӨҖмңјлЎң кё°ліё лқјмқҙлёҢлҹ¬лҰ¬лЎң лҗҳм–ҙмһҲлҠ” Quick Sortм—җ лҢҖн•ҙ мқҙм•јкё°лҘј н•ҙліҙкІ лӢӨ
Cм—җ мһҲлҠ” qsort н•ЁмҲҳлҠ” Quick Sort м•Ңкі лҰ¬мҰҳмңјлЎң кө¬нҳ„мқҙ лҗҳм–ҙмһҲлӢӨ
C++мқҳ кІҪмҡ°м—җлҠ” кё°ліёмқҙ Quick SortмқҙлӮҳ Merge SortмҷҖ к°ҷмқҖ лӢӨм–‘н•ң м •л ¬мқ„ м§Җмӣҗн•ңлӢӨ


Complexity

qsortмқҳ кІҪмҡ°м—җлҠ” ліөмһЎм„ұмқ„ кі л Өн•ҳм§Җ м•ҠлҠ”лӢӨ
sortмқҳ кІҪмҡ°м—җлҠ” C++11мқ„ кё°мӨҖмңјлЎң Worst Caseм—җм„ң O(n logn)мқ„ мҡ”кө¬н•ңлӢӨ
Modern C++(C++20мқҙ лӮҳмҳЁ мӢңм җм—җм„ң C++11мқ„ лӘЁлҚҳмқҙлқјн•ҳлҠ”кІғлҸ„ мқҙмғҒн•ҳлӢӨ) мқҙм „мқҳ кІҪмҡ°м—җлҠ” O(n^2)к№Ңм§Җ н—Ҳмҡ©н–ҲлӢӨ


Performance

qsort мҷҖ sortм—җ лҢҖн•ҙм„ң мӢӨн—ҳн•ң кІ°кіј sortк°Җ лҚ” мҡ°мҲҳн•ң нҚјнҸ¬лЁјмҠӨлҘј ліҙмҳҖлӢӨ


Stability

qsortмқҳ кІҪмҡ°м—җлҠ” void* лЎң м ‘к·јн•ҳкё° л•Ңл¬ём—җ sortмқҳ кІҪмҡ°к°Җ safetyн•ҳлӢӨ
лҳҗн•ң sortлҠ” л°°м—ҙ, STL, class л“ұ лӢӨм–‘н•ң м»Ён…Ңмқҙл„Ҳм—җм„ң м§Җмӣҗмқҙ к°ҖлҠҘн•ҳкё° л•Ңл¬ём—җ нҷңмҡ©лҸ„к°Җ лҶ’лӢӨ
