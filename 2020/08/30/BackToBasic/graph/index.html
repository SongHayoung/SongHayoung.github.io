그래프

프림 알고리즘

  1. 시작 정점을 정하고 그래프에 포함 시킴
  2. 그래프에 포함된 정점의 인접한 정점으로 가는 간선을 모두 가중치가 낮은걸 top에 두는 우선순위 큐에 삽입
  3. 우선순위 큐의 top에 있는 간선 선택, 단 해당 도착 정점이 이미 그래프에 포함되어 있다면 continue

크루스칼 알고리즘

  1. 모든 간선 가중치 오름차순 정렬
  2. 정렬된 간선부터 선택해가며 그래프에 삽입, 단 간선의 양 정점 모두 그래프에 삽입되어 있다면 해당 간선은 선택하지 않음

다익스트라 알고리즘

  1. 시작 정점 선택 및 거리 0으로 초기화
  2. 해당 정점의 인접 정점들 삽입, 간선 가중치 만큼으로 인접 정점 초기화
  3. 삽입된 정점들의 인접 정점들을 계속 탐색해가며 반복, 단 탐색 시 이미 초기화된 정점이 있다면 가중치 비교 후 현재 경로의 가중치가 더 낮다면 현재 가중치의 경로로 변경

프림 알고리즘 vs 다익스트라 알고리즘

  • 프림 알고리즘은 그래프를 최소 비용으로 구성하는게 목적임
  • 다익스트라 알고리즘은 특정 정점에서의 이동 비용을 최소로 구성하는게 목적임

조금 헷갈리면 다음 그래프를 두 알고리즘으로 만들어 보면 차이를 알듯.

시작 정점 A
A - B 2
B - C 2
A - C 3

플루이드 와샬

  • 목적 : 전체 정점에서 다른 정점으로 이동하기 위한 최소 비용 그래프 생성
  1. self to self = 0
  2. 인접 정점으로 가는 간선으로 일단 초기화, 이동 못하면 INF
  3. 결국 요점은 경유해서 가는게 더 빠르냐는거임
