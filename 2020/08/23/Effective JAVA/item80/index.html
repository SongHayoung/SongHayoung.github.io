л“Өм–ҙк°Җл©°

java.util.concurrent нҢЁнӮӨм§Җм—җм„ң к°„лӢЁнһҲ мһ‘м—… нҒҗлҘј мғқм„ұн•  мҲҳ мһҲлӢӨ.

к·ё мҷём—җлҸ„ лӢӨм–‘н•ң кё°лҠҘмқ„ м§Җмӣҗн•ңлӢӨ.

  1. нҠ№м • нғңмҠӨнҒ¬к°Җ мҷ„лЈҢлҗҳкё°лҘј кё°лӢӨлҰ°лӢӨ.
  2. нғңмҠӨнҒ¬ лӘЁмқҢмӨ‘ м•„л¬ҙкІғ н•ҳлӮҳ(invokeAny) нҳ№мқҖ лӘЁл“  нғңмҠӨнҒ¬(invokeAll)к°Җ мҷ„лЈҢлҗҳкё°лҘј кё°лӢӨлҰ°лӢӨ.
  3. ExecutorServiceк°Җ мў…лЈҢн•ҳкё°лҘј кё°лӢӨлҰ°лӢӨ.(awaitTermination)
  4. мҷ„лЈҢлҗң нғңмҠӨнҒ¬л“Өмқҳ кІ°кіјлҘј м°ЁлЎҖлЎң л°ӣлҠ”лӢӨ.(ExecutorCompletionService)
  5. нғңмҠӨнҒ¬лҘј нҠ№м • мӢңк°„м—җ нҳ№мқҖ мЈјкё°м ҒмңјлЎң мӢӨн–үн•ҳкІҢ н•ңлӢӨ.(ScheduledThreadPoolExecutor)

нҒҗлҘј л‘ҳ мқҙмғҒмқҳ мҠӨл Ҳл“ңк°Җ мІҳлҰ¬н•ҳкІҢ н•ҳкі  мӢ¶лӢӨл©ҙ мҠӨл Ҳл“ң н’Җмқ„ мқҙмҡ©н•ҳл©ҙ лҗңлӢӨ. лҚ” м„ём„ён•ң м„Өм •мқ„ мӣҗн•ҳл©ҙ ThreadPoolExecutor нҒҙлһҳмҠӨлҘј мқҙмҡ©н•  мҲҳ мһҲлӢӨ. мһ‘мқҖ н”„лЎңк·ёлһЁмқҙлӮҳ м„ңлІ„лқјл©ҙ Executors.newCachedThreadPoolмқҙ мўӢмқҖ м„ нғқмқҙлӢӨ. л¬ҙкұ°мҡҙ м„ңлІ„лқјл©ҙ мқҙлҠ” мўӢм§Җ лӘ»н•ҳлӢӨ. мғҲлЎңмҡҙ нғңмҠӨнҒ¬к°Җ л“Өм–ҙмҳЁлӢӨл©ҙ мҰүмӢң мҠӨл Ҳл“ңм—җ мң„мһ„лҗҳлҠ”лҚ° к°Җмҡ©н•  мҠӨл Ҳл“ңк°Җ м—ҶлӢӨл©ҙ мҠӨл Ҳл“ңлҘј мғҲлЎң мғқм„ұн•ңлӢӨ. мқҙлҠ” кі§ CPU м җмң мңЁмқ„ 100%лЎң мҳ¬лҰ¬лӢӨк°Җ л–Ём–ҙлңЁлҰ¬лҠ” кІ°кіјлҘј лӮікІҢ лҗҳлҠ”лҚ° л–Ём–ҙм§ҖлҠ” мқҙмң лҠ” context switchingм—җ мһҗмӣҗмқ„ мҶҢлӘЁн•ҳкё° л•Ңл¬ёмқҙлӢӨ. л”°лқјм„ң л¬ҙкұ°мҡҙ м„ңлІ„лқјл©ҙ Executors.newFixedThreadPoolмқҙлӮҳ ThreadPoolExecutorлҘј м§Ғм ‘ мӮ¬мҡ©н•ҳлҠ”кІҢ мўӢлӢӨ.

нғңмҠӨнҒ¬лҠ” Runnableкіј Callableмқҙ мһҲлӢӨ. CallableмқҖ Runnableкіј л№„мҠ·н•ҳм§Җл§Ң к°’мқ„ л°ҳнҷҳн•ҳкі  мһ„мқҳмқҳ мҳҲмҷёлҘј лҚҳм§Ҳ мҲҳ мһҲлӢӨ. к·ёлҰ¬кі  мқҙ нғңмҠӨнҒ¬лҘј мҲҳн–үн•ҳлҠ” мқјл°ҳм Ғмқё л§Өм»ӨлӢҲмҰҳмқҙ ExecutorServiceлӢӨ. нғңмҠӨнҒ¬лҘј ExecutorServiceм—җ л§Ўкё°л©ҙ мӣҗн•ҳлҠ” нғңмҠӨнҒ¬ мҲҳн–ү м •мұ…мқ„ м„ нғқн•  мҲҳ мһҲкі  м–ём ңл“ м§Җ ліҖкІҪмқҙ к°ҖлҠҘн•ҳлӢӨ.

мһҗл°” 7л¶Җн„°лҠ” ExecutorServiceк°Җ fork-join нғңмҠӨнҒ¬лҘј м§Җмӣҗн•ңлӢӨ. fork-join нғңмҠӨнҒ¬лҠ” fork-join poolмқҙлқјлҠ” нҠ№лі„н•ң ExecutorServiceк°Җ мӢӨн–үн•ҙмӨҖлӢӨ. ForkJoinTaskмқҳ мқёмҠӨн„ҙмҠӨлҠ” мһ‘мқҖ н•ҳмң„ нғңмҠӨнҒ¬лЎң лӮҳлү  мҲҳ мһҲкі  FockJoinPoolмқ„ кө¬м„ұн•ҳлҠ” мҠӨл Ҳл“ңл“Өмқҙ мқҙ нғңмҠӨнҒ¬лҘј мІҳлҰ¬н•ҳл©° мқјмқ„ лЁјм Җ лҒқлӮё мҠӨл Ҳл“ңлҠ” лӢӨлҘё мҠӨл Ҳл“ңмқҳ лӮЁмқҖ нғңмҠӨнҒ¬лҘј к°Җм ёмҷҖ лҢҖмӢ  мІҳлҰ¬н•  мҲҳлҸ„ мһҲлӢӨ. мқҙлҘј нҶөн•ҙ CPU мІҳлҰ¬лҹүмқ„ лҶ’нһҲл©° лӮ®мқҖ м§Җм—°мңЁмқ„ лӢ¬м„ұн•  мҲҳ мһҲлӢӨ.
