@Builder

@BuilderлҠ” нҒҙлһҳмҠӨмқҳ ліөмһЎн•ң л№ҢлҚ” APIлҘј мғқм„ұн•ҳлҸ„лЎқ лҸ„мҷҖмӨҖлӢӨ
Person.builder().name("к№Җлҳ”лҳ”").age(26).build();мҷҖ к°ҷмқҙ мӮ¬мҡ©н•  мҲҳ мһҲлӢӨ
@Data @ToString @EqualsandHashCode @Getter @Setter @RequiredArgsConstructorлҘј н•ңлІҲм—җ мӮ¬мҡ©н•ҳлҠ” м–ҙл…ён…Ңмқҙм…ҳмқҙлӢӨ

@Builderм—җ мһҲм–ҙм„ң нҠ№м§•мқҖ лӢӨмқҢкіј к°ҷлӢӨ

  1. л№ҢлҚ”лҠ” non-static non-final н•„л“ңм—җ лҢҖн•ң л©”мҶҢл“ңл§Ң л§Ңл“Өм–ҙмӨҖлӢӨ
    • лӢЁ RequiredArgsConstructorм—җ мқҳн•ҙ мҙҲкё°нҷ”лҗҳм§Җ м•ҠмқҖ final н•„л“ңлӮҳ @NonNull н•„л“ңм—җ лҢҖн•ҙм„ңлҠ” мғқм„ұмһҗлҘј мғқм„ұн•ңлӢӨ
  2. parameterк°Җ м—ҶлҠ” м•Ҫн•ң мғқм„ұмһҗлҘј л§Ңл“Өм–ҙмӨҖлӢӨ

@Builder.Default

н•„л“ңлҘј кі м •мӢңнӮӨл ӨлҠ” мқҳлҸ„лқјл©ҙ final нӮӨмӣҢл“ңлҘј м„Өм •н•ҙм•ј н•ңлӢӨ
н•ҳм§Җл§Ң лӘ…мӢңм ҒмңјлЎң мҙҲкё°нҷ” н•ң л’Ө мғҒнҷ©м—җ л”°лқј к°’мқ„ ліҖкІҪмӢңнӮӨкі  мӢ¶мқҖ кІҪмҡ°к°Җ мһҲлӢӨ
лӘ…мӢңм Ғ мҙҲкё°нҷ”лҘј нҶөн•ң мҙҲкё°нҷ”лҘј м§„н–үмӢңнӮЁ лӢӨмқҢм—җ л№ҢлҚ”лҘј мӮ¬мҡ©н•  кІҪмҡ° лӢӨмқҢкіј к°ҷмқҖ мӢңлӮҳлҰ¬мҳӨлҘј мЈјмқҳн•ҙм•јн•ңлӢӨ

  1. лӘ…мӢңм Ғ мҙҲкё°нҷ”лҘј нҶөн•ң к°қмІҙ л©ӨлІ„ мҙҲкё°нҷ”
  2. л№ҢлҚ”лҘј нҶөн•ң к°қмІҙ н•„л“ң м„Өм •(лӘ…мӢңм Ғ мҙҲкё°нҷ” н•„л“ңлҠ” м„Өм •н•ҳм§Җ м•ҠмқҢ)

мң„мқҳ мғҒнҷ©м—җм„ң л№ҢлҚ”лҠ” нҠ№м • н•„л“ңлӮҳ л§Өк°ңліҖмҲҳм—җ лҢҖн•ң м„ м–ёмқ„ н•ҙмЈјм§Җ м•ҠлҠ”лӢӨл©ҙ 0 / null / falseлҘј м„Өм •н•ңлӢӨ
к·ёл Үкё° л•Ңл¬ём—җ лӘ…мӢңм Ғ мҙҲкё°нҷ”лҘј н•ң л’Ө мғҒнҷ©м—җ л”°лқј к°’мқ„ ліҖкІҪн•ҳкі  мӢ¶мқ„ кІҪмҡ°м—” @Builder.DefaultлҘј нҶөн•ҙ мң„мҷҖ к°ҷмқҙ ліҖкІҪлҗҳлҠ” кІғмқ„ л°©м§Җ н•  мҲҳ мһҲлӢӨ
