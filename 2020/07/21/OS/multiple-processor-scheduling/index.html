вІцВцЉВ▓ўвдгЖИ░ ВіцВ╝ђВЦ┤вДЂ

вІцВцЉ CPUЖ░ђ ВѓгВџЕВЮ┤ Ж░ђвіЦьЋўвІцвЕ┤ Load Sharing(вХђьЋў Ж│хВюа)Ж░ђ Ж░ђвіЦьЋ┤ВДёвІц.

вІцВцЉВ▓ўвдгЖИ░ ВіцВ╝ђВЦ┤вДЂВЌљ вїђьЋю ВаЉЖи╝ в░Ев▓Ћ

Asymmetric MultiProcessing

в╣ёвїђВ╣Г вІцВцЉВ▓ўвдгвіћ Master Server В▓ўвдгЖИ░Ж░ђ вфевЊа ВіцВ╝ђВЦ┤вДЂ Ж▓░ВаЋЖ│╝ ВъЁВХюваЦ В▓ўвдг ЖиИвдгЖ│а вІцвЦИ ВІюВіцьЁюВЮў ьЎювЈЎВЮё В▓ўвдгьЋўЖ▓ї ьЋювІц. вІцвЦИ В▓ўвдгЖИ░вЊцВЮђ ВѓгВџЕВъљВЮў ВйћвЊювДї ВІцьќЅьЋювІц.

Symmetric MultiProcessing

вїђВ╣Г вІцВцЉВ▓ўвдгвіћ Ж░Ђ В▓ўвдгЖИ░Ж░ђ вЈЁВъљВаЂВю╝вАю ВіцВ╝ђВЦ┤вДЂВЮё ВѕўьќЅьЋювІц. вфевЊа ьћёвАюВёИВіцвіћ Ж│хвЈЎ Ready QueueВЌљ ВъѕЖ▒░вѓў Ж░Ђ В▓ўвдгЖИ░ВЮў Ready QueueВЌљ ВА┤ВъгьЋювІц. Ж░Ђ В▓ўвдгЖИ░ВЮў ВіцВ╝ђВЦ┤вЪгЖ░ђ Ready QueueвЦ╝ ьЎЋВЮИьЋ┤ ВІцьќЅьЋа ьћёвАюВёИВіцвЦ╝ ВёаьЃЮьЋ┤ ВДёьќЅьЋювІц.

В▓ўвдгЖИ░ В╣юьЎћВё▒

ьћёвАюВёИВіцЖ░ђ ьі╣ВаЋ В▓ўвдгЖИ░ВЌљВёю ВІцьќЅВцЉВЮ╝ вЋї В║љВІювЕћвфевдгВЌљвіћ Ж░ђВъЦ ВхюЖи╝ВЌљ ВаЉЖи╝вљю вЇ░ВЮ┤ьё░Ж░ђ В▓ўвдгЖИ░ВЮў В║љВІювЦ╝ В▒ёВџ░Ж▓ї вљювІц. ВЮ┤ вЋї ьћёвАюВёИВіцЖ░ђ В▓ўвдгЖИ░вЦ╝ ВЮ┤ВБ╝ьЋўЖ▓ї вљўВЌѕВЮё вЋї вІцВЮїЖ│╝ Ж░ЎВЮђ ВЃЂьЎЕВЮ┤ в▓їВќ┤ВДёвІц.

  1. В▓ўвдгЖИ░ВЮў В║љВІю вЕћвфевдг вѓ┤ВџЕВЮђ вг┤ьџеьЎћ вљювІц.
  2. В▓ўвдгЖИ░ВЮў В║љВІювіћ вІцВІю В▒ёВЏїВаИВЋ╝ьЋювІц.
    В║љВІювЦ╝ вг┤ьџеьЎћьЋўЖ│а вІцВІю В▒ёВџ░віћ ВъЉВЌЁВЮђ в╣ёВџЕВЮ┤ вДјВЮ┤ вЊцЖИ░ вЋївгИВЌљ SMPВІюВіцьЁюВЌљВёю вїђвХђвХёВЮў Ж▓йВџ░ Ж░ЎВЮђ В▓ўвдгЖИ░ВЌљВёю ьћёвАюВёИВіцвЦ╝ ВІцьќЅВІюьѓцвацЖ│а ьЋювІц. ВЮ┤вЦ╝ В▓ўвдгЖИ░ В╣юьЎћВё▒ВЮ┤вЮ╝ ьЋювІц.

вХђьЋў ЖиавЊ▒ьЎћ

SMPВІюВіцьЁюВЌљВёю ВъљВЏљ ВѓгВџЕВЮё ВхювїђьЎћ ьЋўвацвЕ┤ Load BalancingВЮ┤ ВцЉВџћьЋўвІц. Load BalancingВЮђ SMP ВІюВіцьЁюВЮў вфевЊа В▓ўвдгЖИ░ ВѓгВЮ┤ВЌљ вХђьЋўЖ░ђ ЖиавЊ▒ьЋўЖ▓ї вХёВѓ░вљўвЈёвАЮ ВІювЈёьЋювІц. Load BalancingЖИ░віЦВЮђ SMP ВІюВіцьЁюВЮў В▓ўвдгЖИ░в│ё вЈЁвдйВаЂ Ready QueueЖ░ђ ьЋавІ╣вљю ВІюВіцьЁюВЌљвДї ьЋёВџћьЋю ЖИ░віЦВЮ┤вІц. ВЮ┤ Load BalancingВЮђ Processor AffinityВЌљ в░ў(тЈЇ№╝ЅьЋўЖ▓ї вљювІц.

Pull ВЮ┤ВБ╝

Pull ВЮ┤ВБ╝віћ Вюаью┤ВЃЂьЃюВЮў В▓ўвдгЖИ░Ж░ђ в░ћВЂю В▓ўвдгЖИ░вЦ╝ ЖИ░вІцвдгвіћ ьћёвАюВёИВіцвЦ╝ ВъљЖИ░ВфйВю╝вАю PullьЋ┤ВўевІц.

Push ВЮ┤ВБ╝

Push ВЮ┤ВБ╝віћ ьі╣ВаЋ ьЃюВіцьЂгЖ░ђ ВБ╝ЖИ░ВаЂВю╝вАю Ж░Ђ В▓ўвдгЖИ░ВЮў вХђьЋўвЦ╝ Ж▓ђВѓгьЋўЖ│а вХђьЋў вХѕЖиавЊ▒ ВЃЂьЃювЮ╝вЕ┤ вЇю в░ћВЂю В▓ўвдгЖИ░вАю PushвљювІц.

вІцВцЉВйћВќ┤ ьћёвАюВёИВёю

В╗┤ьЊеьё░ ьЋўвЊюВЏеВќ┤ВЮў Ж▓йьќЦВЮђ ьЋўвѓўВЮў вг╝вдгВаЂВЮИ В╣Е ВЋѕВЌљ ВЌгвЪг Ж░юВЮў В▓ўвдгЖИ░ ВйћВќ┤вЦ╝ ВъЦВ░ЕьЋювІц. ВЮ┤вЦ╝ вІцВцЉВйћВќ┤ ьћёвАюВёИВёювЮ╝Ж│а ьЋўвЕ░ Ж╣Ї ВйћВќ┤віћ ЖхгВА░ВаЂВЮИ ВЃЂьЃю ВюаВДђвЦ╝ ВюёьЋю ваѕВДђВіцьё░ ВДЉьЋЕВЮё Ж░ђВДёвІц. ВЮ┤вЦ╝ ьєхьЋ┤ OS ВъЁВъЦВЌљВёю в│ёЖ░юВЮў вг╝вдг В▓ўвдгЖИ░В▓ўвЪ╝ в│┤ВЮ┤Ж▓ї вљювІц. ВЮ┤ вІцВцЉВйћВќ┤ ьћёвАюВёИВёювіћ ВіцВ╝ђВЦ┤вДЂВЮё в│хВъАьЋўЖ▓ї ьЋювІц. Memory StallьўёВЃЂВЮђ Cache Miss вЊ▒ ВЌгвЪг ВЏљВЮИВЌљ ВЮўьЋ┤ в░юВЃЮвљўЖ▓ї вљўвіћвЇ░ ьћёвАюВёИВёюЖ░ђ вЕћвфевдгвЦ╝ ВаЉЖи╝ьЋа вЋї вЇ░ВЮ┤ьё░Ж░ђ Ж░ђВџЕьЋ┤ВДђЖИ░вЦ╝ ЖИ░вІцвдгвЕ┤Вёю ВІюЖ░ёВЮё ьЌѕв╣ёьЋўвіћ ьўёВЃЂВЮ┤вІц. ВЮ┤вЦ╝ ьЋ┤Ж▓░ьЋўЖИ░ ВюёьЋ┤ ьЋўвЊюВЏеВќ┤ ВёцЖ│ёвХђьё░ ВіцваѕвЊюЖ░ђ Ж░Ђ ВйћВќ┤ВЌљ ьЋавІ╣вља Вѕў ВъѕвЈёвАЮ вІцВцЉ ВіцваѕвЊю ьћёвАюВёИВіц ВйћВќ┤вЦ╝ ЖхгьўёьЋювІц.

Ж░ђВЃЂьЎћВЎђ ВіцВ╝ђВЦ┤вДЂ

Ж░ђВЃЂьЎћ ВІюВіцьЁюВЮ┤ ВіцВ╝ђВЦ┤вДЂВЌљ ВўЂьќЦВЮё в»ИВ╣ўвіћ ВџћВєївіћ вДцВџ░ в░ЕвїђьЋўЖИ░ вЋївгИВЌљ ВДДЖ▓ї ьЉюьўёьЋўЖИ░ ьъўвЊцвІц. Ж░ювъхВаЂВю╝вАю Ж░ђВЃЂьЎћ ВєїьћёьіИВЏеВќ┤віћ ВІюВіцьЁюВЌљВёю ВІцьќЅВцЉВЮИ Virtual MachineвДѕвІц ьЋўвѓў ВЮ┤ВЃЂВЮў Ж░ђВЃЂ CPUвЦ╝ ВаюЖ│хьЋўЖ│а вг╝вдг CPUвЊцВЮё веИВІаВЮ┤ ВѓгВџЕьЋа Вѕў ВъѕвЈёвАЮ ВіцВ╝ђВЦ┤ьЋювІц. ВЮ╝в░ўВаЂВЮИ Ж░ђВЃЂ ьЎўЖ▓йВЮђ ьЋўвѓўВЮў ьўИВіцьіИ OSЖ░ђ вІцВѕўВЮў Ж▓їВіцьіИ OSвЦ╝ Ж┤ђвдгьЋўЖ▓ї вљўвіћ ЖхгВА░вАю ВЮ┤вБеВќ┤ВДђЖ▓ї вљўвіћвЇ░ Ж░Ђ Ж▓їВіцьіИ Вџ┤ВўЂВ▓┤Ваювіћ ВІювХёьЋа ьў╣ВЮђ ВІцВІюЖ░ё ВЌ░Вѓ░ВЮё ьЈгьЋеьЋўвіћ ьі╣ВаЋ ВѓгВџЕ ьЎўЖ▓й вЊ▒ ВѓгВџЕВъљВЌљ вДъЖ▓ї ВёИв░ђьЋўЖ▓ї ВА░Вюевља Вѕў ВъѕвІц. ВўѕвЦ╝вЊцВќ┤ 100msВЮё ьЋавІ╣ьЋўвіћ ВІювХёьЋа Вџ┤ВўЂВ▓┤ВаюВЌљВёю Ж░ђВЃЂ веИВІа вѓ┤ВЮў Вџ┤ВўЂВ▓┤ВаюЖ░ђ ВІцВаю Вќ╝вДѕвДїьЂ╝ CPUвЦ╝ ьЋавІ╣в░ЏвіћЖ░ђВЌљ вїђьЋю вѓ┤ВџЕВЮђ Ж░ђВЃЂьЎћ ВІюВіцьЁюВЌљ вІгвацВъѕвІц. ВЋёвг┤вдг Ж░ђВЃЂьЎћ ВІюВіцьЁюВЮў ВіцВ╝ђВЦ┤ ВаЋв│┤вЦ╝ ВЋїЖ│а ВъѕЖ│а ВА░ВюеВЮё ьќѕвІц ьЋўвЇћвЮ╝вЈё Ж░ђВЃЂвеИВІаВЮў ьЃђВЮ┤веИвіћ вг╝вдг CPUВЮў ВІюЖ│ёв│┤вІц Вўцвъў Ж▒ИвдгЖИ░ вЋївгИВЌљ вХђВаЋьЎЋьЋўЖИ░ вЋївгИВЌљ Ж▓йВџ░ВЌљ вћ░вЮ╝ ВіцВ╝ђВЦ┤вДЂВЮ┤ вг┤Вѓ░вља Вѕў ВъѕвІц.
