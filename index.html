Transaction Isolation Level

트랜잭션의 동시성 문제를 예방하기 위해 락을 제어하는 수준

READ UNCOMMITTED (Level 0)

select문이 수행 중 접근하는 모든 데이터에 락이 걸리지 않음

READ COMMITTED (Level 1)

select문이 수행 중 접근하는 모든 데이터에 shared락이 걸림
MYSQL INNODB Default

REPEATABLE READ (Level 2)

한 트랜잭션 내에 모든 select문이 접근하는 모든 데이터에 shared락이 걸림

SERIALIZABLE (Level 3)

한 트랜잭션 내에 모든 select문이 접근하는 모든 데이터에 exclusive락이 걸림
