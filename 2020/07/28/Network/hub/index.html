HUB

허브는 멀티포트 리피터이다. 말 그대로 들어온 데이터를 중계해주는 역할을 수행한다. 허브는 CSMA/CD의 영향을 받게 되는데 즉, 한번에 하나의 노드에서만 데이터를 전송할 수 있다. 허브에 연결된 노드들은 같은 Collision Domain내에 존재하기 때문에 동시에 여러 노드에서 데이터를 전송하게 되면 collision이 발생된다. 허브 두대를 서로 연결하게 되면 하나의 허브처럼 동작하게 되는데 이 때 연결되있는 노드의 수가 더 증가하게 되니 collision이 더욱 발생하게 된다.

Intelligent Hub

인텔리전트 허브는 NMS를 통해서 데이터를 분석 제어 관리를 할 수 있는 허브이다. 또한 문제가 생긴 노드를 검출해 자동으로 Isolation 시켜주는 Auto Partition기능이 있다.
