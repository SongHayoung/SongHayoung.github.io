OSI 7Layer

통신 접속에서 완료까지의 단계를 7계층으로 정의한 규약

Physical Layer

두 장치간 실제 접속에 필요한 물리적 특성에 대한 규칙 정의 

- UTP 케이블


DataLink Layer

물리적으로 연결된 호스트들 사이의 MAC 주소를 활용한 통신 담당 

- 스위치 : cut throught, store and forward 두 가지 방식 모두 지원, 하드웨어적으로 처리  
- 브릿지 : store and forward 방식만 지원, 소프트웨어적으로 처리

cut throught : 패킷의 목적지 주소만 보고 다음 목적지로 전송
store and forward : 패킷을 모두 받고 다음 목적지로 전송

Network

데이터가 원하는 목적지까지 성공적으로 전달되는것을 담당

- 라우터


Transport Layer

프로세스간 통신 담당

- TCP UDP


Session Layer

프로세스가 통신을 관리하기 위한 방법 제공

- SSL, TLS


Presentation Layer

데이터 포멧 확장자 결정

- MPEG, JPEG


Application Layer

사용자와 소프트웨어간 인터페이스 제공

- DNS, FTP, HTTP
