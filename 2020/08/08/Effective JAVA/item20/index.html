[Effective Java] м¶”мғҒ нҒҙлһҳмҠӨліҙлӢӨлҠ” мқён„°нҺҳмқҙмҠӨлҘј мҡ°м„ н•ҳлқј

л“Өм–ҙк°Җл©°

мһҗл°”к°Җ м ңкіөн•ҳлҠ” лӢӨмӨ‘ кө¬нҳ„ л©”м»ӨлӢҲмҰҳмңјлЎң мқён„°нҺҳмқҙмҠӨмҷҖ м¶”мғҒ нҒҙлһҳмҠӨк°Җ мһҲлӢӨ. мһҗл°” 8 л¶Җн„°лҠ” мқён„°нҺҳмқҙмҠӨлҸ„ л””нҸҙнҠё л©”мҶҢл“ңлҘј м ңкіөн•  мҲҳ мһҲкІҢ лҗҳм–ҙ л‘җ л©”м»ӨлӢҲмҰҳ лӘЁл‘җ мқёмҠӨн„ҙмҠӨ л©”мҶҢл“ңлҘј кө¬нҳ„ нҳ•нғңлЎң м ңкіөн•  мҲҳ мһҲлӢӨ. л‘ҳмқҳ к°ҖмһҘ нҒ° м°ЁмқҙлҠ” м¶”мғҒ нҒҙлһҳмҠӨк°Җ м •мқҳн•ң нғҖмһ…мқ„ кө¬нҳ„н•ҳлҠ” нҒҙлһҳмҠӨлҠ” л°ҳл“ңмӢң м¶”мғҒ нҒҙлһҳмҠӨмқҳ н•ҳмң„ нҒҙлһҳмҠӨк°Җ лҗҳм–ҙм•ј н•ңлӢӨлҠ” м җмқҙлӢӨ.

мқён„°нҺҳмқҙмҠӨмқҳ мһҘм җ

мқён„°нҺҳмқҙмҠӨлҠ” лҜ№мҠӨмқё м •мқҳм—җ м•Ҳм„ұл§һм¶ӨмқҙлӢӨ. лҜ№мҠӨмқёмқҙлһҖ нҒҙлһҳмҠӨк°Җ кө¬нҳ„н•  мҲҳ мһҲлҠ” нғҖмһ…мңјлЎң лҜ№мҠӨмқёмқ„ кө¬нҳ„н•ң нҒҙлһҳмҠӨм—җ мӣҗлһҳ мЈјлҗң нғҖмһ… мҷём—җлҸ„ нҠ№м • м„ нғқм Ғ н–үмң„лҘј м ңкіөн•ңлӢӨкі  м„ м–ён•ҳлҠ” нҡЁкіјлҘј мӨҖлӢӨ. мҳҲлҘј л“Өм–ҙ ComparableмқҖ мһҗмӢ мқ„ кө¬нҳ„н•ң нҒҙлһҳмҠӨмқҳ мқёмҠӨн„ҙмҠӨл“ӨлҒјлҰ¬лҠ” мҲңм„ңлҘј м •н•  мҲҳ мһҲлӢӨкі  м„ м–ён•ҳлҠ” лҜ№мҠӨмқё мқён„°нҺҳмқҙмҠӨлӢӨ. мһҗл°”лҠ” лӢӨмӨ‘мғҒмҶҚмқҙ м•Ҳлҗҳкё° л•Ңл¬ём—җ м¶”мғҒ нҒҙлһҳмҠӨлЎңлҠ” лҜ№мҠӨмқёмқ„ м •мқҳн•  мҲҳ м—ҶлӢӨ.

мқён„°нҺҳмқҙмҠӨлҠ” кі„мёөкө¬мЎ°к°Җ м—ҶлҠ” нғҖмһ… н”„л Ҳмһ„мӣҢнҒ¬лҘј л§Ңл“Ө мҲҳ мһҲлӢӨ. нғҖмһ…мқ„ кі„мёөм ҒмңјлЎң м •мқҳн•ҳл©ҙ мҲҳл§ҺмқҖ к°ңл…җмқ„ кө¬мЎ°м ҒмңјлЎң мһҳ н‘ңнҳ„н•  мҲҳ мһҲм§Җл§Ң, нҳ„мӢӨм—җм„ңлҠ” кі„мёөмқ„ м—„кІ©нһҲ кө¬л¶„н•ҳкё° м–ҙл Өмҡҙ к°ңл…җлҸ„ мһҲлӢӨ.

мқҙмҷҖ к°ҷмқҖ кө¬мЎ°лҘј нҒҙлһҳмҠӨлЎң л§Ңл“Өл Өл©ҙ к°ҖлҠҘн•ң мЎ°н•© м „л¶ҖлҘј к°Ғк°Ғмқҳ нҒҙлһҳмҠӨлЎң м •мқҳн•ң кі лҸ„л№„л§Ң кі„мёөкө¬мЎ°к°Җ л§Ңл“Өм–ҙм§„лӢӨ. нқ”нһҲ мЎ°н•© нҸӯл°ңмқҙлқј л¶ҖлҘҙлҠ” нҳ„мғҒмқҙлӢӨ.

лһҳнҚјнҒҙлһҳмҠӨ кҙҖмҡ©кө¬мҷҖ н•Ёк»ҳ мӮ¬мҡ©н•ҳл©ҙ мқён„°нҺҳмқҙмҠӨлҠ” кё°лҠҘмқ„ н–ҘмғҒмӢңнӮӨлҠ” м•Ҳм „н•ҳкі  к°•л Ҙн•ң мҲҳлӢЁмқҙ лҗңлӢӨ. нғҖмһ…мқ„ м¶”мғҒ нҒҙлһҳмҠӨлЎң м •мқҳн•ҳл©ҙ к·ё нғҖмһ…м—җ кё°лҠҘмқ„ м¶”к°Җн•ҳлҠ” л°©лІ•мқҖ мғҒмҶҚ лҝҗмқҙлӢӨ. мғҒмҶҚн•ҙм„ң л§Ңл“  нҒҙлһҳмҠӨлҠ” лһҳнҚј нҒҙлһҳмҠӨліҙлӢӨ нҷңмҡ©лҸ„мҷҖ м•Ҳм •м„ұмқҙ л–Ём–ҙм§„лӢӨ.

мқён„°нҺҳмқҙмҠӨмқҳ нҷңмҡ©

мқён„°нҺҳмқҙмҠӨмқҳ л©”м„ңл“ң мӨ‘ кө¬нҳ„ л°©лІ•мқҙ лӘ…л°ұн•ң кІғмқҙ мһҲлӢӨл©ҙ к·ё кө¬нҳ„мқ„ л””нҸҙнҠё л©”мҶҢл“ңлЎң м ңкіөн•ҙ к°ңл°ңмһҗмқҳ мқјмқ„ лҚңм–ҙмЈјл©ҙ лҗңлӢӨ. лӢӨл§Ң equalsмҷҖ hashCodeк°ҷмқҖ Object л©”мҶҢл“ңл“Өмқ„ л””нҸҙнҠё л©”мҶҢл“ңлЎң м ңкіөн•ҳм§Җ л§җмһҗ.

лҳҗн•ң мқён„°нҺҳмқҙмҠӨлҠ” мқёмҠӨн„ҙмҠӨ н•„л“ңлҘј к°Җм§Ҳ мҲҳ м—Ҷкі  publicмқҙ м•„лӢҢ м •м Ғ л©ӨлІ„лҸ„ к°Җм§Ҳ мҲҳ м—ҶлӢӨ. лӢЁ, JAVA 9 мқҙмғҒ л¶Җн„°лҠ” static privateлҠ” мҳҲмҷёлӢӨ.

мқён„°нҺҳмқҙмҠӨмҷҖ м¶”мғҒ кіЁкІ© кө¬нҳ„ нҒҙлһҳмҠӨлҘј н•Ёк»ҳ м ңкіөн•ҳлҠ” мӢқмңјлЎң мқён„°нҺҳмқҙмҠӨмҷҖ м¶”мғҒ нҒҙлһҳмҠӨмқҳ мһҘм җмқ„ лӘЁл‘җ м·Ён•ҳлҠ” л°©лІ•лҸ„ мһҲлӢӨ. мқён„°нҺҳмқҙмҠӨлЎңлҠ” нғҖмһ…мқ„ м •мқҳн•ҳкі  н•„мҡ”н•ҳл©ҙ л””нҸҙнҠё л©”мҶҢл“ң лӘҮ к°ңлҸ„ н•Ёк»ҳ м ңкіөн•ңлӢӨ. к·ёлҰ¬кі  кіЁкІ© кө¬нҳ„ нҒҙлһҳмҠӨлҠ” лӮҳлЁём§Җ л©”мҶҢл“ңл“Ө к№Ңм§Җ кө¬нҳ„н•ңлӢӨ.
