[기본기 복습] http
2020-09-01|기본기 되새김질하기
Word count: 162|Reading time: 1 min

http

  • stateless 함

request method

  • GET
    • 요청 파라미터를 url에 ?와 함께 보냄
    • Body가 없음
    • 서버에서 어떤 데이터를 조회하기 위함
    • 캐싱 가능
  • POST
    • 바디에 데이터를 실어 보냄
    • 서버에 상태나 값을 추가할 때 사용

https

  • 암호화를 위해 ssl을 사용

ssl

  • 공개키 암호화 + 대칭키 암호화
  • client -> server : hello with random code + supplies encryption algorithms
  • server -> client : hello with random code + choosen encryption algorithms + ssl certificate
  • client : decrypt ssl certificate and get server open key
  • client -> server : encrypt client side random code + server side random with server open key uses encryption algorithm
  • server : confirm data and uses as encryption key

http 동작

  1. 브라우저에 url 입력
  2. 입력된 url을 가지고 DNS에 질의를 통한 ip 주소 획득, 단 없다면 상위 DNS에 가서 질의
  3. ip주소를 가지고 서버 근처 라우터에서 arp 프로토콜을 통해 서버 mac 주소 획득
  4. 서버와 커넥션 맺고 요청 보냄
  5. 서버측에선 요청을 받고 해석해서 요청에 따른 작업 후 결과를 html로 만들어서 전송
  6. 브라우저는 html을 해석해서 브라우저에 띄움
