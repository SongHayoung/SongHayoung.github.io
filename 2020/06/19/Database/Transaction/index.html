Transaction

Database의 상태 변화를 위해 수행하는 작업의 최소 단위이다. 원자성을 가지기 때문에 나눌 수 없다.

ACID 특징

Atomicity(원자성)

트랜잭션은 모두 반영이 되거나 아예 반영이 되지 않아야 한다

Consistency(일관성)

트랜잭션이 성공을 하면 데이터베이스는 일관된 상태를 가져야 한다

Isolation(독립성)

하나의 트랜잭션이 수행중일 때 다른 트랜잭션은 간섭할 수 없다

Durability(지속성)

트랜잭션이 성공하면 결과는 영구적으로 반영되어야 한다
