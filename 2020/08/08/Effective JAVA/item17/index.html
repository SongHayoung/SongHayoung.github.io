[Effective Java] ліҖкІҪ к°ҖлҠҘм„ұмқ„ мөңмҶҢнҷ”н•ҳлқј
2020-08-08|JavaEffective Java
Word count: 696|Reading time: 4 min

л“Өм–ҙк°Җл©°

л¶ҲліҖ нҒҙлһҳмҠӨлһҖ к·ё мқёмҠӨн„ҙмҠӨмқҳ лӮҙл¶Җ к°’мқ„ мҲҳм •н•  мҲҳ м—ҶлҠ” нҒҙлһҳмҠӨлӢӨ. л¶ҲліҖ мқёмҠӨн„ҙмҠӨм—җ к°„м§Ғлҗң м •ліҙлҠ” кі м •лҗҳм–ҙ к°қмІҙк°Җ нҢҢкҙҙлҗҳлҠ” мҲңк°„к№Ңм§Җ м ҲлҢҖ лӢ¬лқјм§Җм§Җ м•ҠлҠ”лӢӨ.

л¶ҲліҖмңјлЎң л§Ңл“Өкё° мң„н•ң лІ•м№ҷ

  1. к°қмІҙмқҳ мғҒнғңлҘј ліҖкІҪн•ҳлҠ” л©”мҶҢл“ңлҘј м ңкіөн•ҳм§Җ м•ҠлҠ”лӢӨ.
  2. нҒҙлһҳмҠӨлҘј нҷ•мһҘн•  мҲҳ м—ҶлҸ„лЎқ н•ңлӢӨ. мҰү мғҒмҶҚмқ„ н•  мҲҳ м—ҶлҸ„лЎқ н•ңлӢӨ.
  3. лӘЁл“  н•„л“ңлҘј finalлЎң м„ м–ён•ңлӢӨ.
  4. лӘЁл“  н•„л“ңлҘј privateлЎң м„ м–ён•ңлӢӨ.
  5. мһҗмӢ  мҷём—җлҠ” лӮҙл¶Җмқҳ к°ҖліҖ м»ҙнҸ¬л„ҢнҠём—җ м ‘к·јн•  мҲҳ м—ҶлҸ„лЎқ н•ңлӢӨ. к°ҖліҖ к°қмІҙлҘј м°ёмЎ°н•ҳлҠ” н•„л“ңк°Җ н•ҳлӮҳлқјлҸ„ мһҲлӢӨл©ҙ л°©м–ҙм Ғ ліөмӮ¬лҘј нҶөн•ҙ л°ҳнҷҳн•ҳлқј.

л¶ҲліҖ нҒҙлһҳмҠӨмқҳ мҳҲ

мң„ нҒҙлһҳмҠӨлҠ” мӮ¬м№ҷм—°мӮ° л©”мҶҢл“ңл“Өмқҙ мқёмҠӨн„ҙмҠӨ мһҗмӢ мқҖ мҲҳм •н•ҳм§Җ м•Ҡкі  мғҲлЎңмҡҙ мқёмҠӨн„ҙмҠӨлҘј л§Ңл“Өм–ҙ л°ҳнҷҳн•ңлӢӨ. мқҙмІҳлҹј н”јм—°мӮ°мһҗм—җ н•ЁмҲҳлҘј м Ғмҡ©н•ҙ к·ё кІ°кіјлҘј л°ҳнҷҳн•ҳм§Җл§Ң н”јм—°мӮ°мһҗ мһҗмІҙлҠ” к·ёлҢҖлЎңмқё н”„лЎңк·ёлһҳл°Қ нҢЁн„ҙмқ„ н•ЁмҲҳнҳ• н”„лЎңк·ёлһҳл°Қмқҙлқј н•ңлӢӨ.

л¶ҲліҖ к°қмІҙмқҳ мһҘм җ

л¶ҲліҖ к°қмІҙлҠ” лӢЁмҲңн•ҳлӢӨ. л¶ҲліҖ к°қмІҙлҠ” мғқм„ұлҗң мӢңм җмқҳ мғҒнғңлҘј нҢҢкҙҙлҗ  л•Ң к№Ңм§Җ к°„м§Ғн•ңлӢӨ. лӘЁл“  мғқм„ұмһҗк°Җ нҒҙлһҳмҠӨ л¶ҲліҖмӢқмқ„ ліҙмһҘн•ңлӢӨл©ҙ к·ё нҒҙлһҳмҠӨлҘј мӮ¬мҡ©н•ҳлҠ” н”„лЎңк·ёлһҳлЁёк°Җ лӢӨлҘё мЎ°м·ЁлҘј н•ҳм§Җ м•Ҡм•„лҸ„ л¶ҲліҖмңјлЎң лӮЁлҠ”лӢӨ.

л¶ҲліҖ к°қмІҙлҠ” к·јліём ҒмңјлЎң thread safeн•ҳм—¬ л”°лЎң лҸҷкё°нҷ”н•  н•„мҡ”к°Җ м—ҶлӢӨ. м—¬лҹ¬ мҠӨл Ҳл“ңк°Җ лҸҷмӢңм—җ мӮ¬мҡ©н•ҙлҸ„ м ҲлҢҖ нӣјмҶҗлҗҳм§Җ м•ҠлҠ”лӢӨ. л¶ҲліҖ к°қмІҙм—җ лҢҖн•ҙм„ңлҠ” к·ё м–ҙл–Ө мҠӨл Ҳл“ңлҸ„ лӢӨлҘё мҠӨл Ҳл“ңм—җ мҳҒн–Ҙмқ„ мӨ„ мҲҳ м—ҶмңјлӢҲ л¶ҲліҖ к°қмІҙлҠ” м•ҲмӢ¬н•ҳкі  кіөмң к°Җ к°ҖлҠҘн•ҳлӢӨ. л”°лқјм„ң л¶ҲліҖ нҒҙлһҳмҠӨлқјл©ҙ н•ңлІҲ л§Ңл“  мқёмҠӨн„ҙмҠӨлҘј мөңлҢҖн•ң мһ¬нҷңмҡ©н•ҳкё°лҘј к¶Ңн•ңлӢӨ.

лҳҗн•ң л¶ҲліҖ нҒҙлһҳмҠӨлҠ” мһҗмЈј мӮ¬мҡ©лҗҳлҠ” мқёмҠӨн„ҙмҠӨлҘј мәҗмӢұн•ҳм—¬ к°ҷмқҖ мқёмҠӨн„ҙмҠӨлҘј мӨ‘ліө мғқм„ұн•ҳм§Җ м•ҠкІҢ н•ҙмЈјлҠ” м •м Ғ нҢ©н„°лҰ¬лҘј м ңкіөн•  мҲҳ мһҲлӢӨ.

л¶ҲліҖ к°қмІҙлҠ” л°©м–ҙм Ғ ліөмӮ¬к°Җ н•„мҡ”м—ҶлӢӨ. м•„л¬ҙлҰ¬ ліөмӮ¬н•ҙлҸ„ мӣҗліёкіј к°ҷмңјлӢҲ ліөмӮ¬ мһҗмІҙк°Җ мқҳлҜёк°Җ м—ҶлӢӨ. к·ёл Үкё° л•Ңл¬ём—җ clone л©”мҶҢл“ңлӮҳ ліөмӮ¬ мғқм„ұмһҗлҠ” м ңкіөн•ҳм§Җ м•ҠлҠ”кІҢ мўӢлӢӨ.

л¶ҲліҖ к°қмІҙлҠ” мһҗмң лЎӯкІҢ кіөмң н•  мҲҳ мһҲмқҢмқҖ л¬јлЎ  л¶ҲліҖ к°қмІҙлҒјлҰ¬лҠ” лӮҙл¶Җ лҚ°мқҙн„°лҘј кіөмң н•  мҲҳ мһҲлӢӨ. мҳҲлҘјл“Өм–ҙ BigInteger нҒҙлһҳмҠӨлҘј ліҙмһҗ. BigIntegerлҠ” лӮҙл¶Җм—җм„ң к°’мқҳ л¶ҖнҳёмҷҖ нҒ¬кё°лҘј л”°лЎң н‘ңнҳ„н•ңлӢӨ. л§Ңм•Ҫ negate л©”мҶҢл“ңк°Җ нҒ¬кё°к°Җ к°ҷкі  л¶Җнҳёл§Ң л°ҳлҢҖмқё мғҲлЎңмҡҙ BigIntegerлҘј мғқм„ұн•ңлӢӨ н•ҳмһҗ. мқҙ л•ҢлҠ” ліөмӮ¬н•ҳм§Җ м•Ҡкі лҸ„ мӣҗліё мқёмҠӨн„ҙмҠӨмҷҖ кіөмң н•ҙлҸ„ лҗңлӢӨ.

л¶ҲліҖ к°қмІҙлҠ” к·ё мһҗмІҙлЎң мӢӨнҢЁ мӣҗмһҗм„ұмқ„ м ңкіөн•ңлӢӨ. мғҒнғңк°Җ м ҲлҢҖ ліҖн•ҳм§Җ м•ҠмңјлӢҲ мһ к№җмқҙлқјлҸ„ л¶Ҳмқјм№ҳ мғҒнғңм—җ л№ м§Ҳ к°ҖлҠҘм„ұмқҙ м—ҶлӢӨ.

л¶ҲліҖ к°қмІҙмқҳ лӢЁм җ

к°’мқҙ лӢӨлҘҙл©ҙ л°ҳл“ңмӢң лҸ…лҰҪлҗң к°қмІҙлЎң л§Ңл“Өм–ҙм•ј н•ңлӢӨ. мқҙ л§җмқҖ л¶ҲліҖмқҙлқјл©ҙ лӢ№м—°н•ҳлӢӨ. н•ҳм§Җл§Ң мқҙ лӮҙмҡ©мқҙ лӢЁм җмқҙ лҗ  мҲҳ мһҲлӢӨ. мҳҲлҘј л“Өм–ҙ 100л§Ң л№„нҠём§ңлҰ¬ BigIntegerм—җм„ң л№„нҠё н•ҳлӮҳлҘј л°”кҝ”м•ј н•ңлӢӨкі  н•ҙліҙмһҗ. к·ёлҹј мқҙлҘј л§Ңл“ңлҠ”лҚ° л„Ҳл¬ҙ нҒ° л№„мҡ©мқ„ мІҳлҰ¬н•ҙм•ј н•ңлӢӨ.

мғҒмҶҚмқ„ л§үкё° мң„н•ң л°©лІ•

мғҒмҶҚмқ„ л§үкё° мң„н•ң л°©лІ•м—” classлҘј finalлЎң м„ м–ён•ҳлҠ” л°©лІ• мҷём—җлҸ„ мғқм„ұмһҗлҘј privateлЎң мғқм„ұн•ҳлҠ” л°©лІ•лҸ„ мһҲлӢӨ. к·ёл ҮкІҢ н•ң л’Ө мқёмҠӨн„ҙмҠӨлҠ” мҠӨнғңнӢұ л№ҢлҚ” нҢЁн„ҙмңјлЎң м ңкіөн•ҳл©ҙ лҗңлӢӨ.

мЈјмқҳмӮ¬н•ӯ

BigIntegerлӮҳ BigDecimalмқҳ мқёмҠӨн„ҙмҠӨлҘј мқёмҲҳлЎң л°ӣлҠ”лӢӨл©ҙ мЈјмқҳн•ҙм•ј н•ңлӢӨ. мқҙ к°’л“Өмқҙ л¶ҲліҖмқҙм—¬м•ј нҒҙлһҳмҠӨмқҳ ліҙм•Ҳмқ„ м§ҖнӮ¬ мҲҳ мһҲлӢӨл©ҙ мқёмҲҳлЎң л°ӣмқҖ к°қмІҙк°Җ м§„м§ң BigIntegerлӮҳ BigDecimalмқём§Җ л°ҳл“ңмӢң нҷ•мқён•ҙм•ј н•ңлӢӨ.

лҳҗн•ң лӘЁл“  нҒҙлһҳмҠӨлҘј л¶ҲліҖмңјлЎң л§Ңл“Ө мҲҳлҠ” м—ҶлӢӨ. л¶ҲліҖмңјлЎң л§Ңл“Ө мҲҳ м—ҶлҠ” нҒҙлһҳмҠӨлқјлҸ„ ліҖкІҪн•  мҲҳ мһҲлҠ” л¶Җл¶„мқ„ мөңмҶҢнҷ”лЎң кҫёлҜёмһҗ. к°қмІҙк°Җ к°Җм§Ҳ мҲҳ мһҲлҠ” мғҒнғңк°Җ мһ‘м•„м§Җл©ҙ к·ё к°қмІҙлҘј мҳҲмёЎн•ҳкё° мү¬мӣҢм§Җкі  мҳӨлҘҳк°Җ мғқкёё к°ҖлҠҘм„ұмқҙ мӨ„м–ҙл“ лӢӨ.

лӢӨлҘё н•©лӢ№н•ң мқҙмң к°Җ м—ҶлӢӨл©ҙ лӘЁл“  н•„л“ңлҠ” private finalмқҙм–ҙм•ј н•ңлӢӨ. лҳҗн•ң мғқм„ұмһҗлҠ” л¶ҲліҖмӢқ м„Өм •мқҙ лӘЁл‘җ мҷ„лЈҢлҗң, мҙҲкё°нҷ”к°Җ мҷ„лІҪнһҲ лҒқлӮң мғҒнғңмқҳ к°қмІҙлҘј мғқм„ұн•ҙм•ј н•ңлӢӨ.
