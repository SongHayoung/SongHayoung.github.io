Abstract ClassмҷҖ Interface

мһҗл°”м—җм„ңлҠ” C++кіј лӢӨлҘҙкІҢ м¶”мғҒ нҒҙлһҳмҠӨмҷҖ мқён„°нҺҳмқҙмҠӨмқҳ к°ңл…җмқ„ лӘ…мӢңм ҒмңјлЎң кө¬л¶„н•ңлӢӨ. C++м—җм„ңлҠ” л‘ҳм—җ лӘ…нҷ•н•ң кё°мҲ м Ғ кө¬л¶„мқҖ м—Ҷкі  лӘЁлҚҳ C++11м—җ мҷҖм„ңм•ј __interface нӮӨмӣҢл“ңк°Җ л№„н‘ңмӨҖмңјлЎң мғқкІјлӢӨ.

кё°ліём ҒмңјлЎң мқён„°нҺҳмқҙмҠӨлҠ” м–ҙл– н•ң н‘ңмӨҖ лҸҷмһ‘мқ„ л§Ңл“Өкё° мң„н•ң н”„л Ҳмһ„мңјлЎң мӮ¬мҡ©лҗңлӢӨ. м—¬кё°м—җ лҚ”л¶Ҳм–ҙ м¶”мғҒ нҒҙлһҳмҠӨлҠ” кө¬мІҙ нҒҙлһҳмҠӨлҘј мң„н•ң мғҒм„ёнҷ”лҗң лҸҷмһ‘л“Өмқ„ лӘ…м„ён•  мҲҳ мһҲлӢӨ. лӢӨл§Ң мһҗл°”м—җм„ңлҠ” лӢӨмӨ‘ мғҒмҶҚмқҙлһҖ к°ңл…җмқҙ м—Ҷкё° л•Ңл¬ём—җ м¶”мғҒ нҒҙлһҳмҠӨлҘј лӢӨмӨ‘мңјлЎң мғҒмҶҚн•  мҲҳ м—ҶлӢӨ. лӢӨл§Ң лӢӨмӨ‘ нҷ•мһҘмқҖ к°ҖлҠҘн•ҳлӢӨ.

н•ҳм§Җл§Ң мһҗл°”8 л¶Җн„° мқён„°нҺҳмқҙмҠӨм—җ default л©”мҶҢл“ңлҘј м¶”к°Җн•  мҲҳ мһҲлҸ„лЎқ кё°лҠҘмқҙ нҷ•мһҘлҗҳл©ҙм„ң м¶”мғҒнҒҙлһҳмҠӨмҷҖ мқён„°нҺҳмқҙмҠӨмқҳ кІҪкі„к°Җ лӘЁнҳён•ҙмЎҢлӢӨ. м¶”мғҒ нҒҙлһҳмҠӨмҷҖ мқён„°нҺҳмқҙмҠӨк°Җ к°Җм§ҖлҠ” м°ЁмқҙлҠ” лӢӨмқҢкіј к°ҷлӢӨ.

мғҒмҶҚкіј нҷ•мһҘ

м•һм„ң м„ӨлӘ…н–Ҳл“Ҝмқҙ м¶”мғҒ нҒҙлһҳмҠӨлҠ” лӢӨмӨ‘ мғҒмҶҚмқҙ л¶Ҳк°ҖлҠҘн•ҳлӢӨ. мқён„°нҺҳмқҙмҠӨлҠ” лӢӨмӨ‘ нҷ•мһҘмқҙ к°ҖлҠҘн•ҳлӢӨ.

мқёмҠӨн„ҙмҠӨ ліҖмҲҳ

мқён„°нҺҳмқҙмҠӨм—җм„ңлҠ” лӘЁл“  ліҖмҲҳлҠ” public static finalмқҙлӢӨ. м¶”мғҒ нҒҙлһҳмҠӨм—җм„ңлҠ” non-static мқҙлӮҳ non-final н•„л“ңлҘј м§Җм •н•  мҲҳ мһҲлӢӨ.

мқёмҠӨн„ҙмҠӨ л©”мҶҢл“ң

мқён„°нҺҳмқҙмҠӨм—җм„ң лӘЁл“  л©”мҶҢл“ңлҠ” public abstractмқҙлӢӨ. м¶”мғҒ нҒҙлһҳмҠӨм—җм„ңлҠ” non-static л©”мҶҢл“ңлӮҳ public protected privateк°ҷмқҖ лӢӨм–‘н•ң Access ModifierлҘј м§Җм •н•  мҲҳ мһҲлӢӨ.
