Copy On Write

л Ҳл””мҠӨ м„ңлІ„ л©”лӘЁлҰ¬ мӮ¬мҡ©лҹүмқҖ мӢӨ лҚ°мқҙн„° нҒ¬кё° + кҙҖлҰ¬ л©”лӘЁлҰ¬ overhead + Copy-on-WriteлЎң мқён•ң м¶”к°Җл©”лӘЁлҰ¬лҘј кі л Өн•ҙм„ң мӮ°м¶ңн•ҙм•јн•ңлӢӨ.

лҰ¬лҲ…мҠӨлҘј кё°мӨҖмңјлЎң мқҙм•јкё°лҘј н•ҙліҙмһҗл©ҙ л¶ҖлӘЁ н”„лЎңм„ёмҠӨм—җм„ң мһҗмӢқ н”„лЎңм„ёмҠӨлҘј forkн•  кІҪмҡ° к°ҷмқҖ л©”лӘЁлҰ¬ кіөк°„мқ„ кіөмң н•ҳкІҢ лҗңлӢӨ. к·ёлҹ°лҚ° мқҙ кІҪмҡ°м—җм„ң л¶ҖлӘЁ н”„лЎңм„ёмҠӨк°Җ лҚ°мқҙн„°мқҳ ліҖкІҪмқ„ мӢңлҸ„н•ҳл©ҙ к°ҷмқҖ л©”лӘЁлҰ¬ кіөк°„мқ„ кіөмң н•  мҲҳ м—ҶкІҢ лҗңлӢӨ. мқҙ л•Ң л¶ҖлӘЁн”„лЎңм„ёмҠӨлҠ” н•ҙлӢ№ нҺҳмқҙм§ҖлҘј ліөмӮ¬н•ң лӢӨмқҢ ліҖкІҪмқ„ мҲҳн–үн•ҳкі  мқҙлҘј Copy On Writeлқј н•ңлӢӨ.

COW л°ңмғқ мӢңм җ

save нҢҢлқјлҜён„°

redis.confм—җ л””нҸҙнҠёлЎң save нҢҢлқјлҜён„°к°Җ нҷңм„ұнҷ” лҗҳм–ҙмһҲлӢӨ. saveлҘј нҶөн•ҙ RDB нҢҢмқјмқ„ мғҲлЎң м“ё л•Ң мһҗмӢқ н”„лЎңм„ёмҠӨк°Җ мғқм„ұлҗҳм–ҙ мһ‘м—…мқҙ м§„н–үлҗҳлҠ”лҚ° мқҙ л•Ң COWк°Җ л°ңмғқн•ңлӢӨ. мҳҲлҘјл“Өм–ҙ л¬јлҰ¬м Ғ л©”лӘЁлҰ¬к°Җ 32GBмқё мӢңмҠӨн…ңм—җм„ң л Ҳл””мҠӨ м„ңлІ„ мқёмҠӨн„ҙмҠӨк°Җ 30GBлҘј мӮ¬мҡ©н•ҳлҚҳ мӨ‘м—җ COWк°Җ л°ңмғқн•ҙм„ң 3GBмқҳ м¶”к°Җ л©”лӘЁлҰ¬к°Җ н•„мҡ”лЎң н•ҙм§„лӢӨл©ҙ real л©”лӘЁлҰ¬к°Җ л¶ҖмЎұн•ҙм„ң Swapмқҙ л°ңмғқн•ҙ м„ұлҠҘ м Җн•ҳк°Җ мқјм–ҙлӮңлӢӨ.

BGSAVE лӘ…л №

BGSAVEлҸ„ лҸҷмқјн•ҳкІҢ мһҗмӢқ н”„лЎңм„ёмҠӨк°Җ мғқм„ұлҗҳм–ҙ мһ‘м—…мқ„ н•ҳкё° л•Ңл¬ём—җ COWк°Җ л°ңмғқн•ңлӢӨ. SAVE лӘ…л №мқҖ л Ҳл””мҠӨ н”„лЎңм„ёмҠӨк°Җ м§Ғм ‘ мҲҳн–үн•ҳкё° л•Ңл¬ём—җ COWк°Җ л°ңмғқн•ҳм§Җ м•ҠлҠ”лӢӨ.

Replication

мһҗмӢ мқҙ Masterмқҙкі  Slaveк°Җ м—°кІ°лҗҳл©ҙ full resyncк°Җ л°ңмғқн•ҳм—¬ RDB нҢҢмқјмқ„ мғқм„ұн•ҳкё° л•Ңл¬ём—җ COWк°Җ л°ңмғқн•ңлӢӨ. мқҙлҠ” ReplicationмӢң нҢҢмқјмқ„ л””мҠӨнҒ¬м—җ м“°м§Җ м•Ҡкі  мҶҢмј“мңјлЎң л„ҳкё°лҠ” мҳөм…ҳмқ„ мӮ¬мҡ©н•ҙлҸ„ лҸҷмқјн•ҳкІҢ л°ңмғқн•ңлӢӨ. Targetмқҙ Diskмқём§Җ Socketмқём§Җм—җ лҢҖн•ң м°Ёмқҙл§Ң мһҲм§Җ forkнӣ„ RDBлҚ°мқҙн„°лҘј л§Ңл“ңлҠ”кұҙ лҸҷмқјн•ҳкё° л•Ңл¬ёмқҙлӢӨ.

auto-aof-rewrite-percentage нҢҢлқјлҜён„°

redis.confм—җ auto-aof-rewrite-percentage 100мқҙлқјлҠ” нҢҢлқјлҜён„°к°Җ мЎҙмһ¬н•ҳлҠ”лҚ° мқҙ нҢҢлқјлҜён„°лҠ” appendonly нҢҢмқјмқҙ 100% м»Өм§Җл©ҙ rewriteн•ҳлҠ” мҳөм…ҳмқҙлӢӨ. RewriteлҠ” AOF мһҗмӢқ н”„лЎңм„ёмҠӨк°Җ forkлҗҳм–ҙ мӢӨн–үлҗҳлҠ”лҚ° мқҙ л•Ң COWк°Җ л°ңмғқн•ңлӢӨ.

BGREWRITEAOF лӘ…л №

BGREWRITEAOF лӘ…л № лҳҗн•ң мһҗмӢқ н”„лЎңм„ёмҠӨк°Җ мғқм„ұлҗҳм–ҙ rewriteлҘј мҲҳн–үн•ҳкё° л•Ңл¬ём—җ COWк°Җ л°ңмғқн•ңлӢӨ.
