Transaction Isolation Level

트랜잭션의 동시성 문제를 예방하기 위해 락을 제어하는 수준

READ UNCOMMITTED (Level 0)

SELECT 문이 수행 중 접근하는 모든 데이터에 락이 걸리지 않음

READ COMMITTED (Level 1)

SELECT 문이 수행 중 접근하는 모든 데이터에 SHARED Lock이 걸림
MYSQL INNODB Default

REPEATABLE READ (Level 2)

한 트랜잭션 내에 모든 SELECT 문이 접근하는 모든 데이터에 SHARED Lock이 걸림

SERIALIZABLE (Level 3)

한 트랜잭션 내에 모든 SELECT 문이 접근하는 모든 데이터에 EXCLUSIVE Lock이 걸림
